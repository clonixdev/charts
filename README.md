Clonix Helm Repo
====================

Helm Charts provided by Clonix 🚀

## 🚀 Installation

To install the charts repository:

```bash
$ helm repo add clonix https://helm.clonix.ar
$ helm repo update
```

To install a specific chart:

```bash
$ helm install some-release clonix/<chart>
```

## Search for available repos

```bash
$ helm search repo
```
